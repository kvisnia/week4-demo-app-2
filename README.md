# DevReady Hello World

This is a sample React application built with [Vite](https://vitejs.dev/).

## Prerequisites

Before you begin, ensure you have [Node.js](https://nodejs.org/) installed on your system.

## Getting Started

1. **Install Dependencies**
   Navigate to the project directory and run:
   ```bash
   npm install
   ```

2. **Run the Development Server**
   Start the app in development mode:
   ```bash
   npm run dev
   ```
   This will start a local server. You can view the app by opening the URL provided in your terminal (usually `http://localhost:5173`).

## Available Scripts

In the project directory, you can run the following commands:

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production to the `dist` folder.
- `npm run preview`: Bootstraps a local static web server that serves the files of the production build in `dist`.
- `npm run test`: Runs the test suite using [Vitest](https://vitest.dev/).

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Vitest](https://vitest.dev/) & [Testing Library](https://testing-library.com/) (for testing)
